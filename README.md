#Details

###demo
crawler.py - クローラー

fbspider.py - facebookをスクレイピング

imgspider.py - 画像をスクレイピング

pagespider.py - htmlをスクレイピング

titlespider.py - タイトルタグをスクレイピング

twispider.py -　twitterをスクレイピング

###parts
imgsave.py - 画像を保存する

input.py - 入力情報をチェックする
